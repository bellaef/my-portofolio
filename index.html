Berikut ini adalah beberapa project yang telah saya kerjakan, baik secara individu ataupun berkolaborasi bersama tim.

OtaVerse

OtaVerse (Otaku Universe) merupakan website e-commerce berbasis Laravel untuk persewaan kostum cosplay se-Tapal Kuda Jatim dengan sistem booking peminjaman, pelacakan, pengembalian, review, dan refund barang. OtaVerse memiliki 3 role user (Otaku, Admin Cosrent, Master Admin) dengan otorisasi dan manajemen akun yang berbeda.

Tech Stack:
  • Laravel (Back-end)
  • Blade + Bootstrap (Front-end)
  • MySQL (Database)
  • JavaScript (Interaktivitas)
  • BinderByte API (Pelacakan Pengiriman)
Jenis Proyek: Individu

Feeling Mood

Feeling Mood merupakan aplikasi android berbasis Flutter yang dapat merubah warna lampu menggunakan arduino berdasarkan ekspresi wajah pengguna dan perubahan cuaca dari Weather API.

Tech Stack:
  • Flutter (Mobile App)
  • Dart (Programming Language)
  • Arduino (Hardware Integration)
  • OpenCV / Face Detection (Ekspresi Wajah)
  • Weather API (Integrasi Cuaca)
  • Bluetooh Serial / USB (Komunikasi ke Arduino)
Jenis Proyek: Tim (Kontribusi: UI/UX & Dokumentasi)
Kunjungi GitHub
